Domusuite® Real Estate Management Tools Wordpress plugin. 

== Description ==

This Wordpress plugin allows all the users of the Domusuite Package to display a list of property in a wordpress page and, with the proper htaccess modifications, it can turn accessible any single property detail page.
To get more information of Domusuite® Real Estate Management Tools visit our web site www.domusuite.com

== Installation ==

1. Update the plugin `domussuite-plugin` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Insert the url of the xml file in the admin page `DomuSuite Options`
